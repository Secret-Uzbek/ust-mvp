> MVP implementation layer for the Universal Semantic Translator: the first
> working surface where UCOM logic becomes an executable product.

## Layer role

This repository is an implementation MVP layer.

It should hold:

- the first working UST implementation;
- practical code and deployable MVP artifacts;
- implementation-facing documentation and release surfaces;
- a readable bridge from specification to working prototype.

It should not become:

- a mojibake helper shell;
- a fake DOI placeholder surface;
- a donor governance repository;
- a mixed archive with no MVP boundary.
